Overview

SilverSat was selected for NASA's 12th round of selections for the CubeSat Launch Initiative. The primary goal of the mission was to provide educational value to middle and high school students through hands-on learning and a real-world work environment. Students work under professional engineers from the Johns Hopkins Applied Physics Laboratory and NASA Goddard to design, build, and operate the CubeSat. SilverSat launched aboard a SpaceX Falcon 9 Rocket on Northrup Grumman's NG-23 ISS resupply mission on September 14, 2025. The satellite is now resting on the ISS until its deployment near the end of November 2025.

Mission Description

The purpose of our satellite is to take pictures of requested locations and upload them to social media in an automated process.

Solution

I developed an end-to-end machine learning pipeline that:

  • Collects and preprocesses customer data from multiple sources
  • Engineers relevant features using domain knowledge
  • Trains ensemble models (Random Forest, XGBoost, Neural Networks)
  • Provides real-time predictions via REST API
  • Generates automated reports and visualizations

Technical Implementation

Data Processing: Used Pandas and NumPy to clean and transform raw data, handling missing values and outliers.

Feature Engineering: Created 50+ features including behavioral patterns, transaction history, and engagement metrics.

Model Training: Implemented cross-validation and hyperparameter tuning using GridSearchCV. Final model achieved 94% accuracy with 0.91 F1-score.

Deployment: Deployed using Docker containers on AWS EC2, with automated CI/CD pipeline.

Results & Impact

  • Achieved 94% prediction accuracy (15% improvement over baseline)
  • Reduced customer churn by 23% through early intervention
  • Processing time reduced from 2 hours to 5 minutes
  • Model now processes 10,000+ predictions daily

Challenges & Learnings

The biggest challenge was handling imbalanced data (only 8% churn rate). I addressed this using SMOTE oversampling and class weight adjustment. This experience taught me the importance of choosing the right evaluation metrics beyond just accuracy.
